# AI Commands

이 폴더는 반복 작업을 표준화한 명령 문서를 모아둔다. 각 문서는 Copilot Agent에 그대로 붙여넣어 실행하는 방식으로 사용한다.

## 사용 방법
1) 필요한 명령 문서를 연다.
2) 전체 내용을 복사한다.
3) Copilot Agent에게 그대로 붙여넣어 실행한다.

## 명령 목록
- rls-audit.md: 현재 RLS 정책을 권한 매트릭스로 점검
- rls-recursion-fix.md: infinite recursion 오류 해결
- supabase-policy-diff.md: 변경된 정책/함수 요약
- slice-new-feature.md: 수직 슬라이스 신규 기능 구현
- bugfix-minimal-diff.md: 에러 로그 기반 최소 변경 수정
- hardening-3step.md: 3단계 하드닝 리팩터링
- dashboard-linking.md: 대시보드 권한별 플로우 연결
- release-check.md: 배포 전 점검 실행
- manual-role-test.md: 역할별 수동 테스트 절차
